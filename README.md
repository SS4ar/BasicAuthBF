# Install
```
$ go get -u github.com/yasserjanah/BasicAuthBruteForcer
```
# Usage
```
$ BasicAuthBruteForcer -url <URL> -user <USER> -passwd <PASSWORD>
```

OR

```
$ BasicAuthBruteForcer -url <URL> -user-file <USER_FILE> -passwd-file <PASSWORD_FILE>
```
